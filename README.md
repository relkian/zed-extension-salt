# Salt (AKA SaltStack) plugin for Zed editor

This is a Salt plugin for Zed editor, heavily based on
[tree-sitter-jinja](https://github.com/cathaysia/tree-sitter-jinja) and Zed
build-in [tree-sitter-yaml](https://github.com/zed-industries/tree-sitter-yaml)
grammars.

Right now it only supports syntax highlighting.
